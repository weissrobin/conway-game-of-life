# Conway's Game of Life / Conwayova Hra života

## Česky

Toto je full-screen implementace **Conwayovy Hry života** pomocí HTML5 Canvas a JavaScriptu.

### Funkce
- Dynamická mřížka přizpůsobující se velikosti obrazovky
- Klasická pravidla Conwaya pro přežití, narození a smrt buněk
- Podpora testování běžných patternů: **Blinker**, **Block**, **Glider**
- Plynulá animace s možností upravit rychlost aktualizace
